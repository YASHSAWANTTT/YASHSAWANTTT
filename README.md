## things i've shipped

**Emma** ‚ÄĒ AI voice agent for language learning inside Zoom. Real-time TTS, multi-language support, meeting-aware context layer. Duolingo but you never leave your class. Served 100+ ASU students.

**MedAgentBench pipeline** ‚ÄĒ Extended Stanford's clinical AI benchmarks with a structured multi-agent system (working memory + question planner + critic + devil's advocate). Took GPT-4.1 Mini from 34% ‚Üí 59% diagnostic accuracy on 310 held-out cases. Turns out the model wasn't the problem ‚ÄĒ the architecture was.

**SOL** ‚ÄĒ AI quiz and grading platform. Built for professors who are tired of grading the same wrong answer 200 times. Pilot stage, university users, actively iterating.
